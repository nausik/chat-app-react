Chat App React
==============

Clone repo:

```
git clone git@github.com:nausik/chat-app-react.git
cd chat-app-react
```

If you don't have `yarn` installed:

```
npm i -g yarn # if not installed
```

Now install dependencies:

```
yarn
```

If you want to build project:

```
yarn build
```

And to run dev build server:

```
yarn dev
```
